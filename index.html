שלב 1 — Open a Guesty Account
Go to guesty.com and open an account
Select בחבילת "Guesty For Hosts" אם יש לך עד 3 properties, או "Guesty" for management מרובה. יש תקופת ניסיון.
Set Up Your Company Details
Company name: Erestel1. Currency: EUR או ILS לפי market העיקרי. אזור זמן.
Get Your Guesty API Key
Settings → API → Generate Key. Save אותו — We'll use it to connect your system.
שלב 2 — Connect Airbnb
בגסטי: Channel Manager → Airbnb → Connect
Guesty תOpen A window of Airbnb. היכנס עם החשבון שלך (או של הלקוח אם הוא Co-host).
Select אילו properties לסנכרן
כל נכס שתSelect — הpriceים, הזfrom Janת וההזמנות יסונכרנו automaticת. שJanי price בגסטי = עדכון instant in Airbnb.
הגדר חוקי תמחור
Guesty → Pricing → Auto Pricing. קבע מינימום (הFloor שלך), מקסימום, וSeasonality. זה בדיוק מה שהמערכת שלך שולטת.
שלב 3 — Connect Booking.com
Channel Manager → Booking.com → Connect
Booking דורשת Property ID ו-API Key. אפשר לקבל דרך Booking.com Extranet → Connectivity → API Credentials.
הגדרת rooms וחבילות
in Booking יש "Room Types". וודא שהם תואמים ל-Guesty. אחרת הסנכרון No יעבוד.
Rate Plans — תוכניות price
Create לפחות 2: Standard Rate ו-Non-Refundable (זול יותר, בלי Cancel). Booking אוהב מגוון.
שלב 4 — Connect Your System to Guesty API
API Endpoint של Guesty
https://open-api.guesty.com/v1/listings
Authorization: Bearer YOUR_API_KEY
Auto price update from your system
PUT /listings/:id/prices
{"basePrice": 115, "minPrice": 70}
Fetch Reservations
GET /reservations?listingId=:id
Returns: all reservations, dates, revenues
שלב 5 — Onboarding לקוח new (5 minutes)
בעל הנכס מוסיף אותך כ-Co-host in Airbnb
Airbnb → Profile → Co-hosts → Add → המייל שלך. הוא נותן לך הרשאת management.
אתה מוסיף את הנכס שלו ל-Guesty
Guesty → Listings → Import from Airbnb → Select את הנכס שלו. 2 minutes.
מגדיר Price Floor וCommission
במערכת שלך: properties → Add נכס → הכנס פרטים + Floor + Commission. All מסונכרן.
